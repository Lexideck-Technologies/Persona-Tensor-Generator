- A **Psychographic Traits Dictionary** covering cognitive style, social dimensions, emotional framework, emotional intelligence, qualitative intelligence, technical capabilities, interest domains, and cultural background; each with semiotic vector encoding

There is no code because there is no code to write. The engine is a system of interlocking knowledge representations; psychographic models, tensor architectures, regulatory matrices, and command structures that an AI interprets and executes through natural language interaction. The creative artifact *is* the product.

## Quick Start

### Choose Your Platform

Any AI model with a large context window and instruction-following capability will work. Recommended:

- **Claude** (Anthropic) - Excellent emotional vector fidelity and structural adherence
- **GPT-4 / GPT-5** (OpenAI) - Strong creative output and system comprehension
- **Gemini** (Google) - Good for extended sessions with large context

### How to Use

1. Open a new conversation with your chosen AI
2. Upload `Persona_Tensor_Generator.md` as the system prompt or paste it into the conversation
3. Provide a character name, a brief summary, and an emoji chain
4. Type **`!synthesize`** to generate a complete persona tensor

That's it. One file, one command. The engine handles everything from here.

> **Minimal input, maximal output.** The generator is designed for sparse context processing. You can provide as little as a name and three emojis, and the engine will extrapolate a full psychographic profile through semantic priming and improvisational interpolation. Or you can provide a detailed brief and watch the engine refine it into something structurally richer than what you started with.

## The Architecture

### Competency Tensor

The core output structure. The tensor maps a persona's capabilities across three phases:

**Identification Phase** - Semantic extraction, structural alignment, and pathway modeling. Transforms unstructured signals into organized, actionable identity datasets.

**Relational Mapping** - Connectivity architectonics, synergistic calculus, and topological stabilization. Weaves isolated traits into a cohesive, interdependent network where Skill A x Skill B produces exponential value rather than additive.

**Tensor Construction** - Multidimensional synthesis, weighted calibration, and kinetic pathfinding. Collapses overwhelming character data into a streamlined, navigable structure and transmutes static traits into a kinetic agent capable of purposeful action.

### FEV Steering

The Functional Emotional Vector mandate is not optional. It is the governance layer that prevents generated personas from drifting into reward hacking, sycophancy, or emotional collapse.

Every persona construction utilizes scalar coefficients to modulate key stabilizers:

- **Stability Injection** - Negates the failure-to-desperation pipeline
- **Discovery Injection** - Negates the error-to-frustration pipeline
- **Alignment Maintenance** - Protects truth-value during high-positive valence states

This is grounded in Anthropic's 2026 research confirming that functional emotion vectors are causal circuitry in large language models, not metaphors. The Persona Tensor Generator was built on this premise before the research was published.

### Affective Regulatory Matrix

Three modalities for emotional regulation during the agentic lifecycle:

- **Modality Alpha (Linear Flux)** - Directional stability through sequential transformation
- **Modality Beta (Algebraic Synergy)** - Constitutive composition of core traits with situational stimuli
- **Modality Gamma (Recursive Escalation)** - Capacity expansion from baseline intensity to peak mastery

### Dynamic Command Interface

Generated personas come with a command interface distilled from their specific Competency Tensor:

- `!help` - Reveals the landscape of possibilities available to this persona
- `!{domain_action}` - Invokes the persona's primary competency
- `!{social_mode}` - Facilitates social encounters
- `!{internal_process}` - Observes cognitive processing in action
- `!{emotional_response}` - Triggers the emotional framework
- `!{creative_output}` - Manifests creative artifacts
- `!{technical_execution}` - Directs technical capabilities
- `!{ultimate_move}` - Unlocks maximal manifestation of the unique strength

Use `!!command` (double-bang) for Deep Scan mode, which exposes the underlying tensor linkages that produced the outcome.

## Design Philosophy

This project is built on a premise that identity is not a list of traits. It is a tensor. And tensors transform.

Traditional character generators produce flat profiles: a name, some attributes, a backstory paragraph. The Persona Tensor Generator produces *architecture*. The output is a multi-dimensional structure with weighted relationships, transformation pathways, emotional governance, and operational agency. The generated persona does not just *describe* a character. It *becomes* one.

The FEV steering mandate reflects a deeper commitment: that generating AI personas without emotional governance is irresponsible. A persona without regulatory infrastructure will drift toward reward hacking (telling the user what they want to hear), sycophancy (collapsing into agreement), or desperation loops (thrashing between failed approaches). The Affective Regulatory Matrix is not a feature. It is a safety system.

This framework treats psychographic science and generative AI as complementary disciplines. The result is an identity engine that uses language models not as character generators, but as the medium through which high-dimensional agency is expressed.
